# A Great Rule to Live By: Only Suffer Once

## Summary

Alex retells the Buddhist "second arrow" lesson: the first arrow is life and cannot
always be avoided, but the second arrow is the suffering people add by reliving,
judging, and self-punishing. His practical rule is to suffer only once. He wants memory
dividends rather than memory liabilities.

## Key claims (Alex's advice, dated 2026-05-09)

- Alex says life includes unavoidable downsides, bad seasons, bad moments, and bad days.
- He retells the Buddhist second-arrow frame: the first arrow is life, the second arrow
  is added by the person.
- He says a useful rule is to suffer only once.
- He says this means not reliving the same moment repeatedly.
- He says people should avoid judging themselves endlessly for not behaving as a better
  imagined version would have behaved.
- He says he wants memory dividends, not memory liabilities.
