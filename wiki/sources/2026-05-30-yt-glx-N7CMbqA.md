# The Actual Definition of Trauma, and How to Stop Using It as an Excuse

## Summary

Alex Hormozi defines trauma behaviorally: a permanent behavior change resulting from an aversive stimulus. He argues that not every bad event is trauma if behavior does not change, and that a behavior change can make someone better or worse relative to long-term goals. He emphasizes that he is not justifying bad events; he is focusing on response and adaptability.

## Key claims (Alex's advice only)

- Trauma should be defined as a permanent change in behavior after an aversive stimulus.
- A bad event is not trauma in this definition unless it changes behavior.
- If behavior changes in a way that supports long-term goals, the person got better from the event.
- If behavior changes against long-term goals, the person got worse from the event.
- Both positive and negative behavior changes can still meet the definition of trauma.
- Alex says he does not justify bad things happening; he treats aversive events as part of life and focuses on response.
- Adaptability determines whether the bad event makes the person better or worse.
